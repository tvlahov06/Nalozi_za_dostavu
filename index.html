Potrebno jednom po uređaju

Upiši svoj GitHub Personal Access Token kako bi se nalozi automatski spremali u bazu. Token se sprema samo lokalno u ovaj browser.

📌 Kako generirati token:
1. Idi na github.com → Settings → Developer settings
2. Klikni Personal access tokens → Tokens (classic)
3. Generate new token (classic)
4. Označi ✅ repo → Generate → Kopiraj token
